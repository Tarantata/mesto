#Научись учиться#

Ссылка на проект: https://Tarantata.github.io/mesto

Это четвёртая работа учебного проекта.
Это первая часть разработки сайта с использованием JavaScript всего будет три части.
По сравнению с предыдущей работой - вёрстка стала проще, основная сложность - 
начало работы с JavaScript. Также предполагается более широкое применение GitHub.

В работе были использованы следующие технологии: структурирование файлов по БЭМ, использование flex и grid контейнеров, адаптивная вёрстка, работа с классами и атрибутами средствами JS, составление функций с помощью JS.

Цель работы - закрепление навыков использования базовых функций и свойств JavaScript.
Для дальнейшего использования работы в учебных целях необходимо провести проверку 
верстки, исправить ошибки, и, по возможности, оптимизмровать код.

Дальнейшая доработка проекта - более широкое использование изучаемых функций Javascript.
